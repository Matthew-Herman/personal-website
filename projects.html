Top down shooter

I'm developing this with Phaser 3, which is an open source JavaScript game engine that I am contributing to. Due to its recent release, Phaser 3's documentation is sparse and the API is constantly changing. I'm planning to upload this as an example on the Phaser 3 example repo because there aren't any good examples showing how to implement these kinds of mechanics.

Controls:

WASD keys: move player

Left click: fire gun

After locking mouse: move the reticle with the mouse

Retro Missile Command

This is a work in progress. I'm developing this with p5.js, which is a port of Processing to JavaScript. Currently this program doesn't work with mobile devices, but I'm planning to add that in later.

Controls:

Arrow Keys: move target reticule

A, S, D Keys: Fire from left, middle, or right missile battery
